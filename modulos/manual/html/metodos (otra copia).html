METODOS PRINCIPALES (general.php)
__CONSTRUCT($option=array())
Este metodo ASIGNA datos a los atributos del objeto, utilizando los metodos secundarios del objeto.
__SAVE($datas=NULL,$option=NULL)
Este metodo CREA O MODIFICA un registro del objeto, utilizando los atributos asignados en el __CONSTRUCT del objeto,
__BROWSE($option=array())
Este metodo FILTRA registros del objeto, utilizando los datos asignados en el objeto,
__DELETE($option=NULL)
Este metodo BORRA registros del objeto, utilizando los datos asignados en el objeto,
__EXECUTE($comando_sql, $option=array("open"=>1,"close"=>1))
Este metodo Ejecuta una instruccion SQL sobre el servidor.
NO SE RECOMIENDA EL USO DIRECTO DE ESTE METODO
